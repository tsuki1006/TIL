## 学び
### jQuery

- アロー関数  

  書き方  `const 名前 = (引数) => { 処理 }`

  ※アロー関数には「自分自身の this を持たず、外側のスコープの this をそのまま引き継ぐ」という性質がある

- 関数 `function() { ... }`と アロー関数の使い分け

- 関数などの引数に`this`は使えない  

  this は特別な意味を持つ言葉（予約語）  
  => `el`（elementの略）や `target`などとする

- this（記事要素）を関数に引数として渡したいとき  
  =>使いたい時点でのthisを定数に保存しておく

  例 `const self = this` 

  使用時の例 `handleHeartDisplay(self, hasLiked)`

### jQuery 要素の取得メソッド

- `find`, `children`

  find => 全ての子要素孫要素  
  children => 直属の子要素

  参考サイト https://popinsight.jp/blog/?p=10873

- `parent`, `parents`, `closest`

  parent => 1つ上の階層である「親要素」を取得  
  parents => すべての親要素を取得

  closest => 親要素内にある**特定の要素**を取得することに特化   
  **※一番最初に合致した要素を1つだけしか取得しない**

  使用例（親要素の中で、id: textの要素を取得）  
  `const elements = $('p').closest('#text')`

  参考サイト https://www.sejuku.net/blog/36261
