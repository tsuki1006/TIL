## 学び

### CSS flex

####  要素１,２,３を「左右端に1と3を置き、2だけ真ん中に寄せず 1 のすぐ隣に置く」やり方  
-**方法１**  

    .wrapper {
    display: flex;
    }

    .item3 {
    margin-left: auto; /* これで3だけ右端に寄る */
    }

-**方法２**　1と2をひとまとめにして、3だけ右に寄せる  
```
<div class="wrapper">
<div class="left">
    <div class="item">1</div>
    <div class="item">2</div>
</div>
<div class="item">3</div>
</div>
```
```
.wrapper {
display: flex;
justify-content: space-between;
}

.left {
display: flex; /* ここで1と2をくっつけて横並び */
}
```

#### flex レイアウトでテキスト省略が効かない時

 flex itemはデフォで`min-width: auto`になり、**縮まらず省略されなくなる**  
→ `text-overflow: ellipsis`が効かなくなる

解決方法：`min-width: 0;` を与える
